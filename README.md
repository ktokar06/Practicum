# Генерация отчетов Allure

### Установка и запуск отчета

Для начала установки Allure выполните команду:

```bash
mvn allure:install
```

Затем соберите отчет командой:

```bash
mvn allure:report
```

И запустите сервер с отчетом следующим образом:

```bash
mvn allure:serve
```
